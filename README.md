# Cat Detection Using YOLO

## Project Overview

This project implements a Cat Detection system using YOLO Object Detection.

The model identifies cats in images, draws bounding boxes around them, and provides confidence scores.

## Dataset

Roboflow Cat Detection Dataset

## Technologies Used

- Python
- YOLO
- OpenCV
- Google Colab
- PyTorch

## Features

- Cat Detection
- Bounding Box Localization
- Confidence Scores
- Multiple Image Testing

## Training Parameters

- Epochs: 50
- Image Size: 640
- Batch Size: 16

## Evaluation Metrics

- Precision
- Recall
- mAP50
- mAP50-95
